# jetsonNano-Tick
Jetson Nano Telegraf InfluxDB Chronograf Telegraf (Tick)

# Deploy tick stack JetsonNano

## Install 
```
git clone git@gitlab.com:raspirepo/jetsonNano-Tick.git
cd jetsonNano-Tick/setup
./install-tick.sh
```

## Start

```
./start-tick.sh
```

## Status

```
./status-tick.sh
```
